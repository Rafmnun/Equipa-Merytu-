A equipa, por dentro.

Perfis de quem respondeu ao questionário, rostos para decorar nomes, e um mapa vivo das relações profissionais — quem trabalha lado a lado, quem se cruza pouco, e onde as funções se ligam.

Cada nó é uma pessoa, colorida pela sua área. As linhas cheias são relações de trabalho próximo (trabalha com / recorre a); as tracejadas são contacto reduzido. Arrasta os nós, passa o rato por cima para ver detalhes, e usa os botões para aproximar. Quem responde puxa as linhas — por isso o centro mostra os núcleos mais ligados.

Arrasta para reorganizar · roda do rato para zoom

Doze respostas ao questionário. Filtra por área para preparar dinâmicas, conversas ou a logística do dia. Os detalhes pessoais (alimentação, limitações) estão visíveis no fim de cada cartão.

Galeria rápida para associar nomes a caras antes do dia. Clica numa pessoa para abrir o seu perfil completo.
